# Viton13 Portfolio

An animated one-page portfolio built for public hosting on GitHub Pages.

Live URL after deployment: `https://viton13bussiness-collab.github.io/`

## Files

- `index.html` - page structure and English portfolio content
- `styles.css` - responsive visual design, motion, and layout
- `script.js` - canvas animation, scroll reveal, and live GitHub repository feed

## Local preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## GitHub Pages

Publish the repository, then enable Pages from the repository root on the `main` branch.
